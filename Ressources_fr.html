Les indispensables

  • Unity : Un des logiciels clef du projet puisqu’il est le moteur de jeu de Zéphyr.
  • Rider : Un des logiciel les plus importants aussi, il nous a permis de réaliser tous les scripts et le site web
  • Overleaf : Ce site internet nous a été indispensable pour faire les rapports LaTeX de haute qualité pour les soutenances et e cahier des charges

    • Ennemis et multijoueurs

  • Navmesh : Permet de gérer les déplacements des intelligences artificielles sur les différentes cartes
  • Photon Unity Network : Ce logiciel permet de gérer les connections entre les différents joueurs
  • Unity Assets Store : Ce site nous a permis de récupérer des Skins et des animations déjà faites, afin que nous puissions nous concentrer sur le développement

    • Design

  • OBS Studio : Il nous a permis d’enregistrer des scènes de gameplay pour le site
  • Bosca Ceoil : Il nous a permis de faire les bruitages du jeu
  • TextMesh Pro : Il nous a permis de faire des jolis menus
  • Bulma : Il s'agit d'un framework css qui a grandement simplifier la création du site web

    • Communication

  • Github : Indispensable pour partager les modifications apportées au projet.
  • Unity Collab' : Nous avons eu un problème avec le repository GitHub et Unity Collab a été utilisé en remplacement
  • Trello : Ce site internet nous a permis d’organiser les différentes tâches de notre projet de façon claire.
  • Discord : Cette application est utile pour faire des appels en groupe et pour nous coordonner.
  • WhatsApp : Cette application nous a permis de communiquer rapidemment

The essentials

  • Unity: One of the key software of the project since it is Zéphyr's game engine.
  • Rider: One of the most important software too, it allowed us to create all the scripts and the website
  • Overleaf : This website was essential for us to make high quality LaTeX reports for the repport and book of specifications

    • Enemies and multiplayers

  • Navmesh : Allows to manage the movements of artificial intelligence on the different maps
  • Photon Unity Network : This software allows to manage the connections between the different players
  • Unity Assets Store: This site has allowed us to find Skins and animations already made, so that we can focus on the development

    • Design

  • OBS Studio : It allowed us to record gameplay scenes for the site
  • Bosca Ceoil : He allowed us to make the sound effects of the game
  • TextMesh Pro: It allowed us to make nice menus
  • Bulma : It's a css framework which has made the creation of the web site much easier

    • Communication

  • Github : Essential to share changes made to the project.
  • Unity Collab': We had a problem with the GitHub repository and Unity Collab was used as a replacement
  • Trello : This website has allowed us to organize the different tasks of our project in a clear way.
  • Discord: This application is useful for making group calls and coordinating us.
  • WhatsApp : This application allowed us to communicate quickly
